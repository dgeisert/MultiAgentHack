python -m loreweaver.main serve
# then open  http://localhost:8000/studio.html

1 · Braindump

Paste freeform notes — world, vibes, half-formed characters, plot scraps. Loreweaver mines them into canon lore + character sheets, merging into the series each time.

2 · Chapter framework

Generate a season-length outline from the world bible. Review it, then produce chapters from it below. Rebuild any time — it won't touch chapters already made.

3 · Chapters

Each framework beat becomes a chapter: authored, QA'd, voiced and published by the agent pipeline. Open one to read its summary, full text, and listen to the audio.
