⚠ Opened as a local file

This page needs the API. Start the server and open it over HTTP:

python -m loreweaver.main serve
# then open  http://localhost:8000/studio.html

1 · Braindump

Paste freeform notes — world, vibes, half-formed characters, plot scraps. Loreweaver mines them into canon lore + character sheets, merging into the series each time.

2 · Chapter framework

Generate a season-length outline from the world bible. Review it, then build chapters from it below. Rebuild any time — it won't touch chapters already made.

3 · Chapters

Open a chapter to build it step by step — pick the characters in it, generate and edit their current states, lay out the plot points, write the prose, cast voices, render the audio line by line, then combine, cover and publish.
